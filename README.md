# 🚀 AI Resume Analyzer & Interview Prep Platform

A full-stack AI-powered career platform built to simulate a real-world product where users can upload resumes, analyze job descriptions, detect skill gaps, generate interview questions, and create ATS-optimized resumes.

This project combines **React**, **Node.js**, **JWT Authentication**, **Gemini AI**, and **Puppeteer** to deliver an end-to-end intelligent hiring preparation system.

---

## 🌟 Features

### 🔐 Authentication & Security
- Secure User Registration & Login
- JWT-based Authentication
- Protected Routes
- Token Blacklisting on Logout
- Session Management

### 📄 Resume Intelligence
- Upload Resume (PDF/DOCX support)
- Resume Parsing & Text Extraction
- Skill Detection from Resume
- Experience & Qualification Analysis

### 📌 Job Description Analysis
- Upload or Paste Job Description
- Compare Resume vs Job Role
- Identify Missing Skills
- Skill Gap Detection using AI

### 🤖 AI-Powered Interview Preparation
- Generate Personalized Interview Questions
- Technical + HR Questions
- Role-Based Question Generation
- Smart Preparation Suggestions

### 📈 ATS Resume Optimizer
- Generate ATS-Friendly Resume
- Improve Resume Keywords
- Optimize Formatting
- Better Recruiter Visibility

### 📄 PDF Generation
- Dynamic Resume PDF Creation using Puppeteer
- Professional Downloadable Resume Output
